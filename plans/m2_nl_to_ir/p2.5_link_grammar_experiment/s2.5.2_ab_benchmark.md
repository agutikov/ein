# S2.5.2 — A/B benchmark + decision

**Phase:** P2.5 (link-grammar experiment).
**Estimate:** 4-5 days.
**Depends on:** S2.5.1.
**Resolves:** [Q11 — link-grammar value](../../open_questions.md#q11--link-grammar-value).

## Context

Quantitative comparison: same LLM, same GBNF, same model file —
two prompt regimes:

- **Baseline** — `grammars/facts.gbnf` + `facts.tmpl`.
- **Augmented** — same + the link-grammar IR fragment inserted into
  the prompt context.

Metric: structural F1 against a hand-curated *gold IR* for each
puzzle in the benchmark set.

## Tasks

### Task T2.5.2.1 — Benchmark puzzle set

`tests/nl/benchmark/`:

- Zebra (canonical, 15 conditions).
- Einstein puzzle variant.
- One additional 5×5 logic-grid puzzle.
- Two "ambiguity-heavy" hand-crafted micro-puzzles (3-5
  sentences with sentence-level ambiguity).

For each: `puzzle.txt` + a hand-curated `puzzle.gold.ein`.

### Task T2.5.2.2 — Metric definition

`src/ein/nl/eval.py`:

```python
def structural_f1(predicted: list[IRNode], gold: list[IRNode]) -> EvalResult:
    """Per-fact precision/recall, treating IR forms as
    multiset members under canonical-print equality (ignoring
    `:source` strings)."""
```

Per-puzzle: precision, recall, F1. Aggregate: macro-average.

### Task T2.5.2.3 — Run + record

`scripts/bench-link-grammar.sh` runs both regimes ≥ 3 seeds each;
saves per-seed predictions under
`tests/nl/benchmark/results/<regime>/<seed>/<puzzle>.ein`. Stable
seeds for reproducibility.

### Task T2.5.2.4 — Decision rule

Document the *pre-registered* decision rule in
`docs/decisions/M2-link-grammar.md`:

- If `F1_augmented - F1_baseline > 0.03` averaged across the 5
  puzzles → ship.
- If `< 0.0` → deprecate.
- In-between → ship as opt-in, default off (`--link-grammar`).

### Task T2.5.2.5 — Document outcome

Write the verdict + numbers into the decision document. Mark
[Q11](../../open_questions.md#q11--link-grammar-value) resolved.

If verdict = deprecate:
- leave `nlp/` in place as a historical reference (no submodule, no move).
- Update `README.md` layout table.
- Remove the `link-grammar` prompt path from `src/ein/nl/`.
