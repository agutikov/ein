# S2.3.2 — Task-specific GBNFs

**Phase:** P2.3 (GBNF for IR).
**Estimate:** 2-3 days.
**Depends on:** S2.3.1.
**Resolves:** [Q24 — one or many grammars](../../open_questions.md#q24--one-gbnf-or-many).

## Context

Working answer for [Q24](../../open_questions.md#q24--one-gbnf-or-many):
**many task-specific GBNFs**. Each restricts the IR kernel to a
single top-level shape, so the LLM doesn't get distracted by forms
it should not emit at this step.

## Tasks

### Task T2.3.2.1 — `grammars/ontology.gbnf`

Restricts top-level forms to `(ontology …)` with `type`,
`instance`, `relation`, `a-priori`, `symmetry` sub-forms. Used by
the ontology-extraction pass (P2.4).

### Task T2.3.2.2 — `grammars/facts.gbnf`

Restricts to a `(facts …)` form containing `(rel …)`,
`(at-position …)`, `(immediately-right-of …)`, etc., with
`:source <string>` mandatory.

The grammar can be *parameterised* on the loaded ontology — only
the declared instance atoms are valid arguments. This task adds
the **template-substitution** layer:

```
;; in grammars/facts.gbnf.in:
nationality-atom ::= "{NATIONALITIES}"
color-atom       ::= "{COLORS}"
...
```

`render_grammar` (P2.2 S2.2.2 T2.2.2.3) interpolates the
ontology's instance lists at request time. So `(rel lives-in Brit Red)`
is rejected if `Brit` isn't an instance of Nationality.

### Task T2.3.2.3 — `grammars/definite-description.gbnf`

For resolving "the green house" → `(the-house-with-color Green)`.
Small grammar: head `the-X-with-Y`, two arguments.

### Task T2.3.2.4 — `grammars/ambiguity.gbnf`

For the parse-time ambiguity wrapper from [Q8](../../open_questions.md#q8--ambiguous-parses):

```
(hypothesis-parse :id <id> :sentence <string>
  :alternatives (<facts-form>+))
```

### Task T2.3.2.5 — Inventory + manifest

`grammars/README.md`: table of grammars, what each restricts the
LLM to, where it's used in the pipeline (P2.4 entry points),
maintenance ownership.

### Task T2.3.2.6 — Tests

`tests/llm/test_task_grammars.py`:

- Each grammar rejects forms outside its sub-language.
- The parameterised facts grammar rejects unknown instance atoms.
- Round-trip: a small hand-written example per grammar parses
  back to canonical IR.

## Acceptance

- `grammars/README.md` documents each.
