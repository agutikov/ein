## Context

Classic CSP heuristic: when picking a value for the current
variable, prefer the value that prunes the fewest options
from neighboring variables. Composes with M1's existing
most-constrained-object first (S1.5.4 A2) — A2 picks the
variable; LCV picks the value.

## Tasks

### Task T1.9.E9.1 — Score function

Per candidate hypothesis: count how many other alive
hypotheses would be killed by this candidate (via the
constraint propagator).

### Task T1.9.E9.2 — Ordering

Sort candidates ascending by score; emit least-pruning first.
