## Implemented (2026-06-15)

Shipped as `inference/min_core.py::minimal_unsat_core(kb)` — but **not** the
re-saturation deletion-MUS the Tasks below describe. That method (drop a core
fact, re-saturate, keep it dropped if still ⊥) is **unsound under this engine's
NAF**: removing a fact can flip an `(absent P)` premise and *fabricate* a
contradiction the full KB never had, so "the subset still contradicts" ≠ "the
subset reproduces the real contradiction." Measured on `zebra2-bad`, that
minimiser degenerates to a structural-declaration residue (relation/property
decls whose removal merely breaks saturation), not the conflict — the same NAF
wall as [E7](s1.9.e7_learned_clause.md)'s clause generalisation and
[E13](s1.9.e13_per_hyp_budget.md).

The sound deliverable, built on [E6](s1.9.e6_transitive_premise_walk.md)'s
`walk_premises`: `KnowledgeBase.unsat_core(witnesses)` unions the source frontier
of *every* contradiction witness, which over-states a single propagating cause.
`minimal_unsat_core` instead returns the **smallest single-witness frontier** —
one *real* derivation's leaves, so it is sound by construction, NAF-safe, and
needs no saturation (pure provenance). On `zebra2-bad`: one injected fact → 123
witnesses → a 38-fact union, but the smallest single-witness frontier is the
**1-fact injected culprit** `(color-loc Green House-1)` — the tightest possible
trace explanation. `tests/inference/test_min_core.py` pins it.

**Use site:** on demand — the P1.6 trace renderer / the contradictions answer
calls `minimal_unsat_core(kb)` for the legible "why" (the verdict keeps the full
`unsat_core` union as the "all facts involved" set). Cheap enough
(provenance-only) to call eagerly if wanted.

## Context (original spec — its re-saturation method is NAF-unsound; see § Implemented)

When a contradiction fires, the unsat-core (premises that
produced it) may be larger than the *minimal* set that
implies the contradiction. Shrinking it (delta-debug or
linear-time minimisation) improves trace readability:
"these N facts together contradict" vs "these M ≫ N facts
include the N that matter".

Pure UX (no correctness gain), but composes with E7
(learned-clause from unsat-core).
