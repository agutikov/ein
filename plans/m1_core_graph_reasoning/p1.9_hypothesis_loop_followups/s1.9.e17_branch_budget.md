## Implemented (2026-06-15)

T1.9.E17.2 (the only open residual): `inference/verdict.py` gains an `Aborted`
verdict (carrying `reason` + the partial stats), and `solve()` gains
`on_budget: "raise" | "verdict"` (default `"raise"`). With `"verdict"` a
`max_enterings` / `max_time` cut **returns** `(Aborted(reason, stats), stats)`
rather than raising `BudgetExceededError`, so a caller needn't catch the
exception. Kept opt-in, and `Aborted` is **out** of the `Verdict` union → the
default raise contract (bench, probes, `test_monotonic_budget`) is unchanged.
Also fixed a latent wart: a budget abort now records `stats.exhausted=False`
(it previously raised before `_finalise_solve` could set it, leaving the
default `True`). `Aborted` is distinct from `Contradiction` — `k=0` here means
*unexplored*, not *proven unsat*. `tests/inference/monotonic/test_monotonic_budget.py`
pins it.

## Already shipped (2026-06-03)

The engine-level budget this entry asks for **already exists**:

- `solve()` (and `gaps_solve` / `contradictions_solve`) take **`max_enterings`**
  (an entering/node-count cap) and **`max_time`** (wall-clock)
  (`inference/monotonic/solver.py:289-290`), threaded into `_explore_layers`
  via `_check_budget`.
- Hitting either raises **`BudgetExceededError`** carrying a reason string +
  the partial `stats` (`solver.py:750-770`); callers catch it (e.g.
  `demo/bench_monotonic.py`, `demo/probe_solution_nodes.py`).
- The wrapper is **`demo/bench_monotonic.py --max-enterings`** (line 205),
  not `bench_solve --max-nodes` — the original wording was stale on both
  counts (wrong script + the flag is `--max-enterings`; "max-nodes" survives
  only as a phrase in its help text).

So **T1.9.E17.1 is effectively done** under the name `max_enterings`. The
genuine residual is **T1.9.E17.2**: today abort is an *exception*; should it
instead (or also) be a **verdict** — an `Aborted` variant or an
`aborted` / `exhausted=False` flag — so a caller wanting a partial answer
needn't catch an exception? (`stats.exhausted` already records whether the
lattice was fully explored, which is part of the way there.)
