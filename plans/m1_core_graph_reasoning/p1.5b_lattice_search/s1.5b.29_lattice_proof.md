## Goal

Ship the `LatticeProof` data class — the lattice engine's
**proof artefact** (the "solution" half of Q1.5b.6's
solution-vs-explanation split). Carries everything P1.6's
NL renderer needs to walk an explanation:

```python
@dataclass(frozen=True)
class LatticeProof:
    """The lattice engine's terminal output —
    the proof artefact per Q1.5b.6.

    P1.6's NL renderer reads this to construct an
    explanation by walking the derivation DAG.
    """
    final_kb:           KnowledgeBase
    # The saturated kb at the satisfying commitment set
    # (or root.kb if no commitment satisfies — coarser).

    lattice:            Lattice
    # The DAG of SetNodes — every visited commitment +
    # state-hash-deduped merges + per-set verdict.

    alive_at_end:       tuple[CanonicalSetId, ...]
    # Commitment sets surviving at termination —
    # candidates for "the solution's commitment".

    dead_commitments:   tuple[DeadCommitment, ...]
    # Refuted commitments + their unsat-cores —
    # the "suppose X then contradiction" narrative
    # source.

    verdict:            Verdict
    # The trichotomy outcome.

    stats:              LatticeStats
    # Cumulative counters — `enterings_total`,
    # `enterings_alive`, `enterings_dead`,
    # `state_hash_merges`, `learned_nogoods_count`,
    # `saturate_count`, `layers_explored`.

@dataclass(frozen=True)
class DeadCommitment:
    canonical_set:  CanonicalSetId
    labels:         tuple[CanonicalSetId, ...]
    unsat_core:     frozenset[Fact]
    learned_clause: frozenset[FactId]
    layer:          int

@dataclass
class LatticeStats:
    enterings_total:       int = 0
    enterings_alive:       int = 0
    enterings_dead:        int = 0
    state_hash_merges:     int = 0
    learned_nogoods_count: int = 0
    saturate_count:        int = 0
    layers_explored:       int = 0
    elapsed_seconds:       float = 0.0
```

## `lattice_solve`'s return shape

Replace the backbone's `Verdict` return with
`LatticeProof`. The trichotomy is still computed but lives
inside `.verdict`; the lattice itself is exposed for
walking.

```python
def lattice_solve(
    root_kb: KnowledgeBase,
    *,
    max_set_size: int = 5,
    config: SolverConfig | None = None,
    mode: Mode = Mode.SOLVE,
    dumper: LatticeDumper | None = None,
) -> LatticeProof:
    """Run the lattice engine; return the proof artefact."""
    ...
    return LatticeProof(
        final_kb=_select_final_kb(lattice, mode),
        lattice=lattice,
        alive_at_end=tuple(
            n.canonical_set
            for n in lattice.nodes_by_set.values()
            if n.verdict in ("alive", "solution")
        ),
        dead_commitments=tuple(
            DeadCommitment(
                canonical_set=n.canonical_set,
                labels=n.labels,
                unsat_core=getattr(n, "unsat_core", frozenset()),
                learned_clause=(
                    n.learned_nogoods[0] if n.learned_nogoods else frozenset()
                ),
                layer=len(n.canonical_set),
            )
            for n in lattice.nodes_by_set.values()
            if n.verdict == "dead"
        ),
        verdict=_synthesise_verdict(lattice, mode),
        stats=stats,
    )

def _select_final_kb(lattice: Lattice, mode: Mode) -> KnowledgeBase:
    """For Solved → the satisfying SetNode's kb.
    For Ambiguity → root.kb (the union of all unconditional
    facts mined across satisfying commitments).
    For Contradiction → root.kb at termination."""
    solutions = [
        n for n in lattice.nodes_by_set.values()
        if n.verdict == "solution"
    ]
    if len(solutions) == 1:
        return solutions[0].kb_snapshot
    # Ambiguity / Contradiction — return the root SetNode's
    # kb (which has accumulated everything reachable).
    root = lattice.get_by_set(())
    return root.kb_snapshot if root else None
```

running in the same process) or `proof/lattice_proof.json`
+ `proof/final_kb.ein` (when consuming a dump).
