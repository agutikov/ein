## Goal

Implement the `--lattice-sanity-check` CLI flag (off by
default) that validates the **saturation commutativity**
premise — the foundation of P1.5b's `d!`-redundancy
elimination claim.

For each multi-parent candidate `C` of size `k ≥ 2`:

1. For each parent `Pᵢ` of `C`:
   - Fork `Pᵢ.kb_snapshot`.
   - Add the missing hypothesis `hᵢ = C \ Pᵢ`.
   - Saturate.
   - Compute `state_hash(fork)`.
2. Assert all `k` resulting state_hashes are equal.

Mismatch ⇒ saturation is NOT commutative on this puzzle's
rule set — either a rule is order-dependent (bug), the rule
set has non-monotone semantics (M1 violation), or the
input was crafted to break the premise (hostile input;
report + abort).

Off by default because the check costs `k × saturation`
per candidate; ship as `--lattice-sanity-check` and run on
demos + zebra2 as a release regression.

## Sanity-check implementation

`inference/lattice/sanity.py` (new):

```python
"""Saturation-commutativity sanity check — Q1.5b.2.d
follow-up. Off by default; enable with --lattice-sanity-check.
"""
from __future__ import annotations

from collections.abc import Iterable
from dataclasses import dataclass

from ein.inference.apriori import CanonicalSetId, FactId
from ein.inference.lattice.lattice import Lattice, SetNode
from ein.inference.saturator import Saturator
from ein.kb.entities import Fact, Layer
from ein.kb.store import KnowledgeBase
from ein.kb.provenance import Provenance

@dataclass
class SanityFailure(Exception):
    canonical_set:   CanonicalSetId
    parents_checked: tuple[CanonicalSetId, ...]
    state_hashes:    tuple[int, ...]   # one per parent

    def __str__(self) -> str:
        return (
            f"Saturation commutativity violated for "
            f"{self.canonical_set!r}: state hashes from "
            f"{len(self.parents_checked)} parents are "
            f"{self.state_hashes}"
        )

def check_commutativity(
    lattice: Lattice,
    canonical_set: CanonicalSetId,
) -> None:
    """For a size-k commitment with k ≥ 2, saturate from
    every parent in turn; assert all state hashes match.

    Raises SanityFailure on mismatch.
    """
    if len(canonical_set) < 2:
        return  # trivially commutative — only one parent

    node = lattice.get_by_set(canonical_set)
    if node is None or not node.parents:
        return

    hashes: list[int] = []
    for parent_set in node.parents:
        parent_node = lattice.get_by_set(parent_set)
        if parent_node is None or parent_node.kb_snapshot is None:
            continue
        # Missing element = C \ parent_set
        missing = tuple(h for h in canonical_set if h not in parent_set)
        if len(missing) != 1:
            continue  # not a layer-(k-1) parent
        fid = missing[0]
        fork = parent_node.kb_snapshot.fork()
        h_fact = Fact(
            relation_name=fid[0], args=fid[1],
            layer=Layer.REASONING,
            provenance=Provenance.from_hypothesis(branch=0),
        )
        stored = fork.add_fact(h_fact)
        fork._index_fact(stored)
        Saturator(fork).saturate()
        from ein.inference.state_dump import state_hash  # or wherever it lives
        hashes.append(state_hash(fork))

    if len(set(hashes)) > 1:
        raise SanityFailure(
            canonical_set=canonical_set,
            parents_checked=node.parents,
            state_hashes=tuple(hashes),
        )
```
