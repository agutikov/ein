### T1.5b.10.1 — Module docstrings audit

  to README.md.

### T1.5b.10.2 — Test surface audit

Walk `tests/inference/monotonic/` and confirm every public
function in `solver.py` + `state_dump.py` has at least one
test exercising it. Cross-reference against the test files
landed in S1.5b.5/.6/.7/.8/.9. Missing coverage gets a new
test in:

- `test_monotonic_unit.py` — per-function unit tests for
  helpers (`_compute_alive`, `_is_solved`, `_emit_negated_fact_writeback`,
  etc.).
- `test_monotonic_integration.py` — small puzzles
  exercising end-to-end loops (alive shrink → re-saturate →
  early-terminate, nogood-emit-and-filter on consecutive
  layers, etc.).

Target: every public + private function in
`monotonic/solver.py` exercised; coverage > 90% via
`pytest --cov`.
