# S1.7a.3 — One search + result-type-from-`k`

**Phase:** P1.7a. **Estimate:** 1–2 days. **Status:** planned.
**Depends on:** [S1.7a.2](s1.7a.2_solution_node_core.md).
**Design:** [`refactoring_design.md` §2–3,5](refactoring_design.md).

## Goal

Make the verdict a **pure function of the run** (`verdict_of(proof)` reading
`k = len(solution_nodes)` + `exhausted`), and retire `is_solved`/`Mode`/`entry`
as the things that decide *search shape* and *termination*.

## Tasks

- **T1 — carry `exhausted` + `stopped_by`.** Add to `LatticeProof`; set in
  `_finalise_*`: an unbudgeted, untruncated run that reached complete-or-dead
  on every branch is `exhausted=True`. (Certification rule from S1.7a.1 T3.)
- **T2 — `verdict_of(proof)`.** Implement the §3 table: `exhausted ∧ k=1` →
  `Solution` (now *means* unique); `∧ k>1` → `Ambiguity`; `∧ k=0` →
  `Contradiction(core=⋃ dead cores)`. `¬exhausted`: `k≥1` → uncertified
  (reuse `Ambiguity` or new `Uncertified`); `k=0` → `Unknown`. Add the two
  `¬exhausted` shapes only if S1.7a.1 shows zebra2 needs them.
- **T3 — drop the `entry`/`mode` decision-making.** Remove `mode = Mode.SOLVE`
  vestige (`solver.py:673`); collapse the `entry`-dispatched verdict synthesis
  into the single `verdict_of`. The loop now always records solution nodes +
  dead commitments; the *reading* is post-hoc. `is_solved` no longer
  terminates (kept only as `goal_bindings` matcher for S1.7a.6 if needed).
- **T4 — keep the three entries working (Option A shims).** Re-express
  `monotonic_solve`/`gaps_solve`/`contradictions_solve` as thin wrappers over
  the unified `search` + `verdict_of`, narrowing the return type to the
  historical shape so the existing suite stays green in this commit. Mark them
  deprecated; full rename deferred.

## Acceptance

- `verdict_of` unit-tested against synthetic proofs for all five rows.
- zebra2 exhaustive → `Solution` with `exhausted ∧ k=1` (the **unique** model,
  not the partial dead-end). zebra2-minus-15 → `Ambiguity` (`k>1`).
  zebra2-bad → `Contradiction` (`k=0`, exhausted, real core).
- **Invariant test:** no SAT variant → `Contradiction`; no UNSAT → `Solution`.
- Existing suite green via shims; `ruff` green.
