## Goal

Surface the sound result to users: an English answer line for the Unique
verdict, and a verdict-consuming trace. This is the user-visible end of the
refactor and feeds the [idea-08 trace](../../../docs/ideas/08-human-style-deductive-trace.md).

## Tasks

- **T1 — CLI `solve` answer path.** `ein-bot solve <file>` runs the unified
  `search` + `verdict_of`. On **Unique**, render the goal projection as words
  ("The Japanese keeps the zebra; the Norwegian drinks water."), exit 0. On
  **Ambiguity**, report `k` + the residual open set. On **Contradiction**,
  report the unsat core. `--mode {solve,gaps,contradictions}` selects the
  preset (S1.7a.5 T2); reconcile with the current `--mode {gaps,contradictions}`
  (`cli.py:377`).
- **T2 — trace linearizer.** `trace/linearize.py` consumes the new verdict:
  spine = the unique model's firings; reductios = dead commitments; closing
  grid from the solution node. Confirm it still renders the idea-08 checklist
  moves (the S1.6.5 gate must stay green).
- **T3 — answer rendering helper.** Small `trace/answer.py` (or reuse if the
  S1.7.3 experiment left one) that turns goal bindings + the relation
  semantics into the English line.
