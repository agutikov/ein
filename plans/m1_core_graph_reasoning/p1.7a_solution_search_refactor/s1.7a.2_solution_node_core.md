# S1.7a.2 — `solution_node` core: complete ∧ consistent, deduped

**Phase:** P1.7a. **Estimate:** ½–1 day. **Status:** **shipped 2026-05-31** — `ein.py/src/ein/inference/solution.py` (`is_solution_node` / `open_hypotheses` / `complete` / `consistent`); `solver._compute_alive` delegates to it (commit `4297deb`).
**Depends on:** [S1.7a.1](s1.7a.1_investigations.md) findings (esp. OQ1).
**Design:** [`refactoring_design.md` §1–2](refactoring_design.md).

## Goal

Introduce the domain-agnostic *solution node* and wire it into the lattice
recording path **behind the existing entries** (no verdict change yet) — so
the predicate is exercised and tested before the API flips in S1.7a.3.

## Tasks

- **T1 — predicates.** Add `open_hypotheses(kb)` / `has_open_hypothesis(kb)` /
  `complete(kb)` / `consistent(kb)` / `is_solution_node(kb)` (home:
  `inference/solution.py` or `verdict.py`). Make `_compute_alive`
  (`solver.py:393`) the single implementation that `open_hypotheses` wraps
  (or alias it) — no second copy. Built only on `generate_hypotheses` +
  pos/neg fact sets (no `is-a`/`total`/signatures).
- **T2 — `SolutionNode` type + dedup map.** Add `SolutionNode`
  (state_hash, per-branch snapshot kb, witnessing commitment, firings) to
  `monotonic/lattice.py`; add `solution_nodes_by_hash` to the loop state and
  `solution_nodes: tuple[...]` to `LatticeProof` (deduped view alongside the
  raw `solutions` list).
- **T3 — record on `is_solution_node`, not `is_solved`.** In `_explore_layers`,
  alongside (not yet replacing) the existing `is_solved` recording, populate
  `solution_nodes_by_hash` whenever `is_solution_node(result.kb)`. Dedup by
  `state_hash`.
- **T4 — tests.** `is_solution_node` true for a completed zebra2 model kb,
  false for the grid[0] dead-end; `state_hash` dedup collapses zebra2's 14
  winners to 1; the incomplete dead-end is absent from `solution_nodes`.

## Acceptance

- New predicates unit-tested; `_compute_alive` not duplicated.
- On zebra2 (gaps, exhaustive), `len(proof.solution_nodes) == 1` while the
  legacy `len(proof.solutions) == 15` (both present — the dedup is proven by
  the divergence). No verdict/CLI behaviour changed yet. `ruff` green.
