# S1.20.A0 — Reconcile kernel docs against the live kernel (pre-flight)

**Phase:** P1.20 (kernel documentation), pre-flight (cross-theme).
**Estimate:** 1-2 days — an audit + a batch of cheap factual fixes
(the per-theme reconciles themselves live in B / C / D / I).
**Status:** parked; **run first.** Re-bases the rest of P1.20.
**Depends on:** the kernel surface being stable enough to document —
[P1.7a](../p1.7a_solution_search_refactor/) (sound solution model) and
the flat-form grammar ([P1.7c](../p1.7c_block_head_removal/) Track A).
**Blocks:** every other P1.20 theme — they assume an accurate
starting state, and several were written assuming an *empty* or
*stub* one.

## Context

P1.20 was authored 2026-05-24 (themes) / 2026-06-15 (stages). Between
those dates and M1's close the implementation surface moved
substantially, so several themes describe a kernel that no longer
matches the code or the existing docs. This stage **diffs
`docs/kernel/` against the live kernel** and fixes the factual drift
*before* any new authoring, so B/C/D/I don't plan against a phantom.

The drift axes found in the 2026-06-16 P1.20 review:

| # | drift | live truth | themes affected |
|---|-------|-----------|-----------------|
| 1 | **Block-head grammar** | flat-form since P1.7c — `(ontology …)`/`(facts …)`/`(reasoning …)`/`(rules …)` wrappers are GONE; a fact is any head outside the closed declarator set `{relation, rule, hrule, query, config, trace, macro, import}`; layer is per-fact | **B** (kernel API), any "`(rules …)`/`(facts …)` context" phrasing |
| 2 | **One sound `solve`** | the `monotonic_solve` / `gaps_solve` / `contradictions_solve` trio was merged into a single `solve()`; the verdict is read from `k` (0/1/>1), never chosen by the entry point | **D** (inference docs), README examples |
| 3 | **Stdlib shipped** | `ein.py/src/ein/stdlib/` carries `algebra`/`bijection`/`closure`/`elim`/`typing`/`macro` (P1.8a S1.8a.f20), imported by `examples/zebra2.ein` | **C** (stdlib API — now unblocked) |
| 4 | **Inference docs grew** | `docs/kernel/inference/` is ≈1.7k lines across 5 files (incl. `architecture_and_algorithms.md`), but its `README.md` is internally stale (stub banner, never-built `01–05` "planned structure", "not yet" table) | **D** (reconcile, not greenfield) |
| 5 | **Config flags churned** | the lattice rewrite removed `enable_alive_inherit` / `enable_eager_root_bubble` / `enable_path_condition_nogoods`; the live levers are a different set in `config.py` | **I** (feature × config matrix) |
