# S1.8.A5 — Migration to imports

**Phase:** P1.8 (improvements), Theme A.
**Estimate:** ½-1 day, depending on how much zebra*.ein
content moves.
**Status:** parked; terminal stage of Theme A's import chain.
**Depends on:** [S1.8.A4](s1.8.a4_rule_library_location.md).
**Blocks:** nothing; informs [S1.10.C](../p1.10_kernel_docs/s1.10.c_stdlib_api_reference.md)
once shipped.

## Context

After S1.8.A4 ships the library, M1's puzzles (zebra2.ein and
the branching demos) should migrate to use imports instead of
inlining the kernel rules. This stage's scope depends on the
S1.8.A4 decision:

- **If (a) or (c):** zebra2 → `(import "stdlib/kernel.ein")`
  for the kernel rules; keep `cross-attr-spatial-*` and other
  puzzle-specific rules inline.
- **If (b) inline-only:** this stage is empty.

## Tasks

### Task T1.8.A5.1 — Inventory zebra2.ein

What rules in zebra2.ein are universal (move to stdlib) vs
puzzle-specific (stay inline)?

- Universal: `symmetric`, `transitive`, `includes`,
  `disjunctive-prune`, possibly `cross-attr-spatial-fwd/bwd`
  if generalised.
- Puzzle-specific: condition-(N)-derived activator facts;
  zebra2-specific spatial-rule activators.

### Task T1.8.A5.2 — Migrate zebra2.ein

- **a.** Replace the universal `(rule …)` definitions with
  `(import "stdlib/…")`.
- **b.** Keep the puzzle-specific activator facts.
- **c.** Re-run `bench_solve` — same verdict + bindings as
  pre-migration.
- **d.** Trace + node count: match pre-migration (modulo any
  trivial reordering).

### Task T1.8.A5.3 — Migrate branching demos

- **a.** Same pattern for `examples/branching/*.ein` that
  use universal rules.
- **b.** Re-run demos; node counts match pre-migration.

### Task T1.8.A5.4 — Documentation

- **a.** Update `examples/README.md` to point at
  `stdlib/README.md`.
- **b.** Each migrated demo includes a comment at top:
  `;; Imports stdlib (file: examples/stdlib/…)`.

## Acceptance

- zebra2 + demos use imports for universal rules.
- Pre/post-migration verdicts + bindings + node counts
  identical.
- Documentation updated.
